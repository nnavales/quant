# Quant

A very personal finance app. Track your transactions, accounts, net worth and KPIs
from a desktop app, with an optional Telegram bot to log expenses by text or
photos of receipts.

## Features

- Transactions with categories, channels and accounts
- Net worth and historical tracking
- Dashboard with financial KPIs
- Dollar rates and macro data
- Telegram chatbot assistant (text + receipt images)
- Backups and theming

## Stack

- **App:** Tauri 2 + React + TypeScript (Vite)
- **Backend:** Go
- **DB:** SQLite
